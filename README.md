# Funny Shader

Мини-проект на Taichi: анимированный шейдер с плотной сеткой цветных колец на черном фоне.  
Реализация находится в `main.py` (`CircleGridShader`).

<img src="main.gif" alt="Shader preview" style="width: 100%; height: auto;">

## Что делает шейдер

- Рисует повторяющуюся сетку колец (SDF + domain repetition).
- Для каждой ячейки выбирает случайный размер/цвет через hash.
- Анимирует пульсацию, мерцание и движение колонок вниз с разной скоростью.

## Структура

- `main.py` — основной шейдер и запуск.
- `gui.py` — базовый рендер-цикл (`BaseShader`).
- `core.py` — мат. утилиты и hash/smoothstep.
- `sdf.py` — SDF-примитивы и операции.
- `colors.py` — палитры и функции цвета.

## Быстрый старт

```bash
pip install taichi
python3 main.py
```

Закройте окно, чтобы остановить рендер.
